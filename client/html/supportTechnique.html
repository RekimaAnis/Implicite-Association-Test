Problèmes pour effectuer le TAI

Quel type de navigateur est nécessaire pour prendre les tests ?

Votre système doit avoir les spécifications suivantes :

- Javascript autorisé
- Cookies autorisés
- Fenêtres pop-up autorisées
- Flash Player Plugin (version 6.0 ou plus récente) de Adobe est requis pour effectuer les tests. Si vous n’êtes pas sûrs de la version de Flash que vous avez sur votre ordinateur, vous pouvez vérifier en cliquant ici.

Les utilisateurs de Linux doivent avoir les caractères communs installés. WebTV n’est pas compatible avec les tests pour le moment. Les utilisateurs de Macsne doivent pas utiliser Internet Explorer.

Si le problème persiste, veuillez signaler la nature des difficultés rencontrées.
